# Vrinda Store Annual Sales Dashboard (Excel) - 2022

## Project Overview
This project contains an interactive Excel dashboard that analyzes the sales performance of Vrinda Store for the year 2022. The dashboard provides a comprehensive view of sales trends, customer demographics, order status, and channel-wise performance.
## Key Features
- *Orders vs Sales:** Monthly trend analysis combining total sales amount and order counts.
- *Sales by Gender:** Distribution of sales between men and women customers.
- *Order Status:** Breakdown of orders by status such as Delivered, Cancelled, Refunded, and Returned.
- *Top 5 States:** Bar chart showing sales performance across top five states.
- *Orders by Age vs Gender:** Segmentation of orders by customer age groups and gender.
- *Sales by Channel:** Visualization of sales distribution across multiple sales channels like Amazon, Flipkart, Meesho, Myntra, etc.
- *Dynamic Filtering:** Interactive slicers to filter data by Month, Channel, and Category for detailed analysis.
